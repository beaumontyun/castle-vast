## Castle Vast Landing Page

##### Visit http://www.castle-vast.com

##### Courtesy of Nangialai Stoman for the Vue template.

1. ##### Install packages and dependencies:

```
npm install
```

2. ##### Start a local dev server at `http://localhost:8080`:

```
npm run serve
```

3. ##### Build

```
npm run build
```

4. ##### Deploy
```
surge dist\
```
